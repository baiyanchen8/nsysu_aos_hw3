# Mood Diary (心情日記)

這是一個基於 Flutter 開發的跨平台心情日記應用程式。
核心設計理念為「本地優先 (Local-First)」，確保使用者隱私，並結合 AI 技術提供即時的情緒回饋（雞湯語錄）。

## ✨ 主要功能 (Features)

*   **📅 日曆視圖**: 直觀查看每月的日記狀態與心情分佈。
*   **📝 富文本編輯**: 支援 Markdown 語法與插入圖片，記錄生活點滴。
*   **😊 心情追蹤**: 紀錄每日心情與具體 Emoji，視覺化情緒變化。
*   **🤖 AI 雞湯回饋**:
    *   **本地模式**: 內建精選語錄庫。
    *   **AI 模式**: 支援串接 OpenAI / Google Gemini / Local LLM (LM Studio)，根據日記內容生成專屬安慰。
*   **📦 資料管理**:
    *   **備份與還原**: 支援匯出包含圖片的 ZIP 備份檔，輕鬆遷移資料。
    *   **批量管理**: 方便清理或整理舊日記。
*   **📚 語錄擴充**: 可匯入外部 JSON 擴充雞湯庫。

## 📂 資料夾結構說明 (Project Structure)

本專案的主要程式碼位於 `flutter_app1` 資料夾中：

*   **`flutter_app1/`**: Flutter 專案根目錄。
    *   **`lib/`**: 應用程式核心程式碼。
        *   `data/`: 資料層，包含資料庫模型 (ObjectBox)、備份服務、AI 服務邏輯。
        *   `ui/`: 介面層，包含所有頁面 (Screen) 與共用元件 (Widget)。
        *   `providers/`: 狀態管理層 (Riverpod)，負責跨頁面狀態共享。
    *   **`assets/`**: 靜態資源檔案 (如預設的 `quotes.json` 雞湯語錄)。
    *   **`android/`, `ios/`, `windows/`, `linux/`**: 各平台的原生配置檔案。

本專案的雞湯位於 `json_file` 資料夾中:
- 內部有不少`雞湯.json`
- 後續應該也會繼續更新
